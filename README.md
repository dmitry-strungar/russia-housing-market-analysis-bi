# BI-дашборд: рынок жилой недвижимости России

## Анализ цен, факторов стоимости и региональных различий

---

## Обзор проекта

Данный проект представляет собой аналитический дашборд в Power BI, разработанный для исследования рынка жилой недвижимости России.

Дашборд позволяет анализировать структуру рынка, выявлять ключевые факторы, влияющие на стоимость объектов, а также сравнивать регионы по основным показателям.

---

## Бизнес-контекст

Рынок недвижимости характеризуется высокой неоднородностью: стоимость объектов зависит от множества факторов, включая площадь, количество комнат, этаж и регион.

Для эффективного принятия решений необходим инструмент, позволяющий:

- Оценивать текущую ситуацию на рынке  
- Выявлять драйверы стоимости  
- Сравнивать регионы  
- Анализировать структуру предложения  

---

## Цели проекта

- Провести комплексный анализ рынка недвижимости России  
- Определить ключевые факторы, влияющие на цену  
- Выявить региональные различия  
- Построить интерактивный дашборд  

---

## Аналитическая логика

В рамках проекта применялись:

- Агрегированные показатели (средняя цена, цена за м², количество объектов)  
- Группировка данных (по площади, этажам, количеству комнат)  
- Фильтрация и сегментация данных  
- Обработка аномалий и выбросов  

---

## Ключевые инсайты

- **Средняя цена объекта составляет ~5,7 млн руб., при этом цена за м² - около 95-110 тыс. руб.**  
  Это отражает общий уровень цен на рынке жилой недвижимости.

- **Объекты с 1–2 комнатами составляют более 60% всех предложений.**  
  Рынок ориентирован на массовый сегмент.

- **Цена за м² снижается при увеличении площади:**  
  небольшие квартиры (до 40 м²) имеют максимальную стоимость за метр,  
  тогда как крупные объекты демонстрируют более низкую цену за м².

- **Разница между регионами превышает 2–3 раза по средней цене.**  
  Это указывает на сильную региональную дифференциацию рынка.

- **ТОП-10 регионов формируют основную часть предложения.**  
  Рынок характеризуется географической концентрацией.

- **Этаж и тип здания оказывают влияние на стоимость, но не являются ключевыми факторами.**

- **Выявлены аномалии в данных:**  
  нулевые цены, экстремально высокие значения, а также объекты вне территории РФ.

## Структура дашборда

Отчет состоит из трех страниц:

2. Анализ факторов стоимости  
3. Региональный анализ  

---

## Модель данных

Модель данных включает одну таблицу:

- Объекты недвижимости с характеристиками:
 - Дату объявления   
 - Цену   
 - Площадь, площадь кухни в м²   
 - Количество комнат  
 - Этаж, этажность дома   
 - Тип здания, тип объекта   
 - ID региона, ID улицы, ID дома   
 - Координаты объектов   


Дополнительно реализованы:

- Расчет показателя **цена за м²**  
- Группировки по площади и этажности
