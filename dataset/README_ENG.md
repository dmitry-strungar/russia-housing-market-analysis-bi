# Dataset

The original dataset is not included directly in this repository.

## Reason

The data file is significant in size and is located on an external source.

## Data Source

The dataset was obtained from Kaggle:

Link: https://www.kaggle.com/datasets/mrdaniilak/russia-real-estate-2021

## Data Description

The dataset contains information about residential real estate in Russia, including:

 - Listing date  
 - Price  
 - Area, kitchen area in m²  
 - Number of rooms  
 - Floor, number of floors in the building  
 - Building type, property type  
 - Region ID, Street ID, House ID  
 - Property coordinates  

## Note

To reproduce the analysis, download the dataset from the link above and connect it in Power BI.
