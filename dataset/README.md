# Данные проекта

В данном репозитории отсутствуют исходный датасет и файл Power BI (.pbix).

## Причина

Файл проекта Power BI превышает 100 МБ, что выходит за пределы ограничений GitHub на размер файлов.

По этой причине проект не может быть опубликован напрямую в репозитории.

## Источник данных

Датасет взят с платформы Kaggle:

👉 Ссылка: https://www.kaggle.com/datasets/mrdaniilak/russia-real-estate-2021

## Описание данных

Датасет содержит информацию об объектах жилой недвижимости в России, включая:

 - Дату объявления   
 - Цену   
 - Площадь, площадь кухни в м²   
 - Количество комнат  
 - Этаж, этажность дома   
 - Тип здания, тип объекта   
 - ID региона, ID улицы, ID дома   
 - Координаты объектов   

## Примечание

Скриншоты дашборда представлены в папке `/screenshots`, что позволяет ознакомиться с результатами анализа.
